Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

To use the sticky position, set the position property to sticky and define a top value determining where it will stay.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

I used the top value of 10px to keep the box from overshadowing the other box but being very close to it and then I did 80px left to extend it further out to the right side.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

I used the value of 500px for the top property to put the box towards the bottom and then 10px for right.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used 50px for the bottom property and it positions it further from the bottom while the left attribute puts it further right.

  • Hexadecimal codes use base-16 numbers from 00 to FF (0 to 255 decimals) which is composed of red green and blue respectively, RR, GG, BB. This provides a compact code and is widely supported, but is hard to read and change manually.
  • RGB codes just use a decimal value from 0 to 255 in r,g,b format. This can make it easy to fine tune a color but is hard to adjust brightness or saturation without a tool.
  • HSL codes use Hue(360 degrees), Saturation(0-100%), and Lightness(0-100%). This can be even better than rgb to adjust colors manually but is not as supported in old browsers and isnt as compact.

I prefer to use HSL codes because they are easy to manipulate.

HWB summary - Hue, Whiteness, Blackness: A color model to define the color, amounts of white, and black.

CMYK summary - Cyan, Magenta, Yellow, and Key(Black): Not supported in html but is suggested for designers to specify colors more accurately
